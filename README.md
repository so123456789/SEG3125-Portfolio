SEG3125 Portfolio – University of Ottawa

Assignment 1 Reflection & Future Directions

The primary goal of this first assignment was to establish a clean, functional portfolio entry page that I can iteratively expand throughout the semester. While the current version presents my core identity and work, I plan to add more pages (e.g., a dedicated contact page) to imrpove navigation and user engagement.

Looking ahead, I intend to introduce background animations to increase visual interest without distracting from content. I will also enhance interactivity, for instance, adding micro‑interactions that respond to user actions. The color palette may evolve as I progress through this project, ensuring it remains both personal and professionally appealing.

One immediate improvement I will make is taking a higher‑resolution photo of myself and replacing the current photo. I also plan to frame it more thoughtfully, perhaps using a custom border or soft shadow to better align with the principles of visual hierarchy and balance discussed in the NN/g videos.
